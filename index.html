IoT Smart Restroom

Build devices to monitor restroom environment.

  • Date: September-December 2019
  • Responsibility: Frontend Development

Use humidity, VOC(volatile organic compound) sensors and switches (for restroom occupancy) to monitor the envrionment of restroom. Connect sensors and switches with arduino like the picture above. Configure ESP32 to facilitate host communication between Arduino and ESP32. The hardware directly post data through ESP32 to backend, where the data will be stored in database for analysis. Frontend get data from backend api.

Backend provides api for hardware to post and frontend to get. In addition to the instant data, backend also calculate the average value for each kind of data and stored into the database. Frontend will get history data and plot line chart for the data in the last 24 hours.

The frontend development include three parts. History data visualization, monitor for instant data, and current weather and forecast. Users can check the monitor to know restroom occupancy, air quality and history data. React Framework is used to update each section independently, where the instant data is updated every 5 seconds and the weather and history data is updated one hour a time.
